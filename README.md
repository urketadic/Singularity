# Singularity

### Running locally

Start by cloning this repo and making sure you have node and npm installed


**Setup**
```npm install```

If you get npm errors, try globally installing the packages needed:
```npm install -g surge rimraf budo```

**Run dev server**
```npm start```

**Deploy**
```npm run deploy```
